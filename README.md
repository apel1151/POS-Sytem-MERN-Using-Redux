# POS-system

POS mern stack project with react, redux, express, nodejs, mongodb
usecases:

1. user login/register
2. user dashboard
3. item filtering
4. add item
5. delete item
6. edit item
7. generating customer
8. bills calculation
9. invoicing and print
10. logout
