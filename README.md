# POS-system

## POS mern stack project with react, redux, express, nodejs, mongodb
<h1>Usecases</h1>

1. user login/register
2. user authentication with jwt
3. user dashboard
4. item filtering
5. add item
6. delete item
7. edit item
8. generating customer
9. bills calculation
10. invoicing and print
11. logout
